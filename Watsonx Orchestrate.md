## Watsonx Orchestrate 
> Concept
- 자연어: 일상언어, 인간의 언어
<br/><br/>
> Manage Agent

`Agent style`
- Default? ReAct?

`Knowledge`
- Agent가 참조할 지식 추가
- 문서 기반: PDF, DOC, TXT, 웹링크, 업로드 문서...

`Toolset`
- Tools: 외부기능
    - REST API 호출, DB 조회, 이메일 발송, Slack 메시지 전송...
- Agents: 다른 Agent 호출?

`Guidelines`
- 행동 제약, 응답 정책
    - codition: 조건
    - Action: 조건이 참일 때 수행 동작

`Chat with documents`
- 특정 문서를 대상으로 직접 질의?

`Behavior`
- Role, Categories, Rules, Output Format... 정의

`Channels`
- Agent가 배포되는 경로
    - Draft: 개발
    - Live: 운영

`Build with AI`
- Agent를 설계, 구성
- 자연어로 "이런 에이전트를 만들고 싶다" 입력

`Talk to agent`
- Agent를 실행, 테스트
- 실제 사용자처럼 질문 입력
